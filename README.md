# AGE Digitals - Professional Portfolio Website

A modern, fully responsive portfolio website for Atunbi Gabriel Ebunoluwa, showcasing web development services, projects, and credentials.

## Features

✅ **7 Complete Pages:**
- Home (index.html)
- About
- Projects
- Services
- Process
- Credentials/CV
- FAQ

✅ **Fully Responsive Design:**
- Mobile-first approach
- Tablet optimized
- Desktop enhanced

✅ **Modern Technologies:**
- Pure HTML5 (semantic markup)
- Custom CSS3 (no frameworks)
- Vanilla JavaScript (no dependencies)
- CSS Grid & Flexbox layouts
- CSS Variables for theming

✅ **Key Features:**
- Animated hero section with code visualization
- Interactive FAQ accordion
- Smooth scrolling navigation
- Mobile hamburger menu
- Contact form with email integration
- Social media integration
- WhatsApp integration
- Professional animations and transitions

## Setup Instructions

1. **Replace Images:**
   - Replace `img/about.jpg` with your professional photo
   - Replace `img/hero_image.jpg` with your preferred hero image
   - Recommended size: 400x400px minimum

2. **Update Contact Information:**
   - Email: atunebig@gmail.com (already configured)
   - WhatsApp: +234 904 838 3179 (already configured)
   - Update social media links in footer

3. **Customize Content:**
   - Update project details in `projects.html`
   - Add real project images
   - Update service descriptions if needed

4. **Deploy:**
   - Upload all files to your web hosting
   - Maintain the directory structure
   - Ensure all paths are correct

## Customization Guide

### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #0ac930;
    --primary-dark: #089025;
    --primary-light: #0ef03a;
}
```

### Typography
The website uses Google Fonts:
- DM Serif Display (headlines)
- Work Sans (body text)

### Animations
Adjust animation timing in JavaScript (`js/main.js`) or CSS (`css/style.css`)
